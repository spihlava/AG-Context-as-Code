---
description: Start a new feature track (Architect)
---
1. Create a directory `.antigravity/tracks/{args}`
2. Read `.antigravity/roles/architect.md` (adopt role)
3. Read `.antigravity/context/product.md` and `.antigravity/context/stack.md`
4. Create `.antigravity/tracks/{args}/spec.md` with:
   - Boundaries
   - Data Model
   - Acceptance Criteria
